refactoring.ro
==============

Install
======

Add this to your composer.json

"repositories": [
        {   
            "type": "vcs",
            "url": "https://github.com/khelonium/refactoring.ro"
        }   
    ],  
    "require": {
        "php": ">=5.3.3",
        "zendframework/zendframework": ">2.2.0rc1",
        "ro.refactoring": "*" 
    }
