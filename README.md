refactoring.ro
==============

Install
======

Add this to your composer.json

```json
"repositories": [
        {   
            "type": "vcs",
            "url": "https://github.com/khelonium/refactoring.ro"
        }   
    ],  
    "require": {
        "php": ">=5.4",
        "ro.refactoring": "*" 
    }   

```
